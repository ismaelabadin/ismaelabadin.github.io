StudyUp

In this project, I developed a Java web app with Java Server Pages (JSP), which allowed me to use HTML, CSS, and JavaScript to build the website itself, while invoking Java code to run in the backend. The primary function of the app was to allow students to create events at specific coordinates on an interactive map such that other nearby students could meet up and study, hence the name StudyUp.
